# Aprendizaje de PySide6 🚀

Este repositorio contiene una colección de ejemplos y ejercicios prácticos para aprender **PySide6**, el conjunto oficial de enlaces de Python para el framework Qt.

## 📁 Estructura del Proyecto

El proyecto está organizado en las siguientes secciones:

### 🔹 Ejemplos de PySide6 (`ejemplos-pyside6/`)
Aquí encontrarás scripts numerados que cubren desde conceptos básicos hasta componentes avanzados:
- **Introducción y Señales**: Configuración inicial, `QLabel`, `QPushButton` y el sistema de señales/slots.
- **Widgets Básicos**: `QCheckBox`, `QRadioButton`, `QComboBox`, `QSlider`, `QProgressBar`, `QSpinBox`.
- **Arquitectura**: Separación de lógica, patrón MVC (Modelo-Vista-Controlador).
- **Diálogos y Ventanas**: `QDialog`, `QMessageBox`, `QMainWindow`.
- **Interfaz Avanzada**: `QActions`, `QToolBar`, `StatusBar`, `TreeView`.
- **Utilidades**: Validadores, habilitar/deshabilitar controles dinámicamente.

### 🔹 Conceptos Adicionales (`recursividad/`)
- **Recursividad**: Un ejemplo práctico sobre el uso de funciones recursivas en Python.

## 🛠️ Requisitos

Para ejecutar estos ejemplos, necesitarás:

1.  Python 3.10+
2.  PySide6

### Instalación de dependencias

```bash
pip install PySide6
```

## 🚀 Cómo empezar

Cada archivo está diseñado para ser ejecutado de forma independiente. Por ejemplo, para ejecutar el primer ejemplo:

```bash
python ejemplos-pyside6/"01 introducción.py"
```

## 📝 Notas de aprendizaje
Este repositorio sirve como bitácora personal en el proceso de dominio de interfaces gráficas con Python y Qt6.
